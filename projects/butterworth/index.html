Butterworth Filter:

A 3rd-order Butterworth filter with a Sallen-Key and 1st-order cascade.


GITHUB
DOWNLOAD

Concept:

A 3rd-order Butterworth filter is a maximally flat filter with 60dB roll-off. This implementation is derived from a cascade of a 2nd-order active filter and a 1st-order passive filter.
